# script.skynet.colorpicker

***June 7,2026: 60+ more colors added to the palette***

Kodi color picker palette based on the original one but with some new added functionality. This one is compatible with Kodi Omega. I saw a lot of reports online about the original colorpicker addon breaking after Python 3 was introduced. I have been using Estuary Mod V2 skin for a few years now and even though it allows for significant customization...it still relies on the stock "native" Kodi color palette. 

I don't know if this color picker will work in other skins. But it works in my slightly modded version of Kodi Omega Estuary Mod V2 skin which is also available on my GitHub page.
